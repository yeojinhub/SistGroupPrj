## 🖥️ 개요
>
> 본 프로젝트는 SIST 웹사이트에서 수집된 사용자 검색 로그 데이터를 기반으로, 사용 키워드, 브라우저 종류, 응답 코드 통계 등 다양한 정보를 정량·정성적으로 분석하는 Swing 기반 로그 분석 애플리케이션입니다.
>
> 인증 시스템과 파일 선택 기능(JFileChooser)을 포함하고 있으며, 관리자 권한에 따라 보고서 자동 생성 기능까지 제공합니다.
>
